Wat is VIM?
VIM is een sneltoets-gedreven tool waarmee je razendsnel code schrijft, navigeert en bewerkt, zonder muis. In het begin is het wennen, maar deze site helpt je op weg.
Leer hier meer over VIM en het spel The VIM Trails.
